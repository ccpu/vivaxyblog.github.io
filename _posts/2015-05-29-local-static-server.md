适用于前端开发的静态资源服务器。

- 基于nodejs。
- 支持get和post请求。
- 端口被占用时，自动换端口启动服务。
- 支持livereload，`here -w`开启，监听html，js，css文件变化，并自动刷新。

安装方式：

`sudo npm install -g serve-here`

用法：

在项目目录下`here`
